Write code in angular-2-exercise/app.

Run npm start to:

  1. compile your code in /app on change
  2. open a browser to a live-reloading page

  3. if you don't see what you expect, right click and click 'Inspect'. Then open the console tab

  4. we want to make a component that'll control the display of a dataset

  5. define a new component called ChartControl, which binds to <chart-control> elements
  6. add in an input type=range in its template, and some text
  7. add <chart-control> into the template of App
  8. what happens? Think about why
  9. we need to inform Angular we're using a custom component.
  10. import ChartControl into App
  11. create a directives array in App's component definition, and place ChartControl in that array
  12. you should now see your ChartControl component

Input Output

Write code in ./exercises/app.

Run npm start to:

  1. compile your code in /app on change
  2. open a browser to a live-reloading page

  3. if you don't see what you expect, right click and click 'Inspect'. Then open the console tab

  4. passing in values

    1. in App, define a maxAge property and assign 24 * 60 * 60 * 1000 to it
    2. using template syntax, pass in maxAge to the ChartControl
    3. you should see Angular complain that maxAge is not a "known native property"
    4. why? we'll fix this in the next step
  5. having our component accept input
    1. we need to tell Angular our chart-control component has an input for that type
    2. in ChartControl, define an input called maxAge. Type it as a number
    3. we'd like to see the value of our input for debugging
    4. add a ngOnChanges(changes) method, and console.log(changes)
    5. bind the maxAge property to the range's max= attribute. It should set itself on load
  6. listening to output
    1. we want to listen to filtered events from ChartControl in App, and handle them
    2. listen to a filtered event on the chart-control element in App, handling it and passing the $event variable into a new App method called setFilter
    3. you should see Angular complain that filtered is not a "known native property"
    4. why?
    5. we need to tell Angular we're emitting filtered events
    6. in ChartControl, define an output called filtered. It's an EventEmitter. Use the cheatsheet to remind yourself how
    7. define setFilter as a simple console logger - we'll get it to fire later
  7. firing events
    1. we want to fire filtered events each time the user configures our ChartControl via our maxAge slider
    2. we need to store the state of our filter - define a filter propert and initialize it with maxAge: 0
    3. listen to the change event of our input in ChartControl. handling it with a setMaxAge(maxAgeInput.value). You'll want to define a local variable maxAgeInput - use the cheat sheet
    4. in setMaxAge(age: number), call updateFilter({ maxAge: age })
    5. define updateFilter(changes) as a method that
      1. uses Object.assign to copy the current filter state, update it with changes, and reassign it to filter
      2. then uses emit on our filtered output, passing the new filter as the event
    6. when you move the maxAge slider, you should see your setFilter handler in App fire

Built-in directives

Steps

  1. we want to show an unknown number of products
    1. we're going to define a component to display a list of products
      1. define a component - you can choose the name
      2. this should accept a list of products - define an @Input to accept them
      3. use a built-in directive to create a <product> tag per item in the product list
      4. look up a loop-y directive in the cheatsheet
      5. inside the tempate use *ngFor to repeat our product component
      6. pass the product to the <product> component via its input
    2. use our product list to display a list of products
      1. in App, import a longer list products - it's longerStaticProductList from app/productList.fixture.
      2. replace our repeated <product> with the <product-list>
      3. pass in the products to our list component
    3. wohoo, more products with less typing!

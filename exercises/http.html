HTTP

Write code in ./exercises/app.

Run npm start to:

  1. compile your code in /app on change
  2. open a browser to a live-reloading page

  3. if you don't see what you expect, right click and click 'Inspect'. Then open the console tab

  4. creating a service to load reddit data

    1. define a Reddit.service class, with a frontpage method
    2. import and ask for Http in the constructor and define it as a property
    3. import your new service into App, and ask for an instance of it in the constructor
    4. you should get an error - Angular doesn't know how to inject Reddit yet!
    5. to do this we need to do 2 things
      1. tell Angular how to inject Reddit - import and use @Injectable() on Reddit
      2. give Angular a provider for Reddit. In App define a providers: array on the @Component, and import and pass in Reddit
    6. you should now have a reddit property in App
  5. loading data
    1. we want to implement frontpage in Reddit
    2. make a get request to https://www.reddit.com/.json (yes the suffix looks weird - try it out in the browser)
    3. use toPromise() on the result
    4. log out the response via .then((resp) => console.log(resp.json()))
    5. we need to return a transformed version of the response, with just the interesting data
    6. in the .then() handler, use .map to get just the .data properties of the story data, and return it from .then

  6. triggering our load

    1. in App, assign this.stories to the result of calling reddit.frontpage

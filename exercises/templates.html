Templates

Write code in ./exercises/app.

Run npm start to:

  1. compile your code in /app on change
  2. open a browser to a live-reloading page

  3. if you don't see what you expect, right click and click 'Inspect'. Then open the console tab

  4. use this (cheatsheet) for reference - there's lots of syntax!

  5. displaying data
    1. we want to display some data in our template
    2. define a constructor for your App component
    3. assign a string property with a name of your choice
    4. use the interpolation syntax to display it in the template
    5. also try out the interpolation syntax out with the title= attribute of an element. Hover over it to see the browser title popup - does the interpolation work?
  6. configuring elements from data
    1. we want to use Angular to change the color of an element, controlled by a property
    2. define a property holding an HTML colour string
    3. use the attribute syntax to bind that to the color style attribute of an element in your App template
  7. responding to user input
    1. we want to define a + and - button that makes an element's text size grow and shrink
    2. define a fontSize attribute on App
    3. bind it to the font-size style property of an element
    4. define a changeTextSize(delta: number) method on App
    5. create two <button> elements, one with a + and one -
    6. use event listener syntax to listen to click events, with positive and negative deltas
